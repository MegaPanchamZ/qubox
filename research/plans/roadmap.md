# Research Roadmap

## Active

- Transport architecture research for production-grade Qubox streaming.
- Whole-system upgrade planning for transport, receive pipeline, viewer runtime, input, and platform backends.

## Next

- Convert research findings into an implementation plan for transport, input, and platform modules.
- Execute Phase 0 instrumentation and Phase 1 packet transport redesign.

## Milestones

- M1: Measurement baseline and runtime instrumentation.
- M2: Packetized media transport and feedback protocol.
- M3: Client receive pipeline with reorder, assembly, and FEC.
- M4: Viewer runtime replacement with proper scaling and window modes.
- M5: Input subsystem rewrite with controller support.
- M6: Native capture and acceleration backends.
- M7: Production hardening and GUI integration.
